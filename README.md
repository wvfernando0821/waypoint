# Waypoint

An AI-assisted platform for migrating legacy desktop applications (.NET
WinForms, VB6, Java Swing/NetBeans) to modern web apps — analysis, build,
and deployment, with a human approval gate before anything reaches
production.

- **[legacy-migration-platform-spec.md](legacy-migration-platform-spec.md)** — full technical specification
- **[ROADMAP.md](ROADMAP.md)** — phased build plan (milestones M0–M9)
- **[migration-platform-ui-prototype.html](migration-platform-ui-prototype.html)** — clickable UI mockup (open directly in a browser)
- **[engine/](engine/)** — M1: the AI analysis engine proof-of-concept (in progress — see `engine/README.md`)
